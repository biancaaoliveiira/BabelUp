# 🌍 BabelUp - Sistema de Gerenciamento de Idiomas

O **BabelUp** é uma aplicação desktop desenvolvida em Python para gestão de cursos de idiomas. O sistema permite desde o cadastro de alunos até a contratação de planos e evolução de níveis, com uma interface amigável e persistência de dados.

## 🚀 Funcionalidades (CRUD Completo)
- **Create (C):** Cadastro de novos usuários e novos contratos de idiomas no banco de dados.
- **Read (R):** Consulta de catálogo de idiomas, autenticação de login e carregamento de perfil.
- **Update (U):** Atualização automática do nível do aluno conforme progresso e alteração de planos contratados.
- **Delete (D):** Função de encerramento de conta com exclusão dos registros do usuário.

## 🛠️ Tecnologias e Requisitos Técnicos
- **Linguagem:** Python 3 (Variáveis, Estruturas Condicionais, Laços e Funções).
- **POO:** Programação Orientada a Objetos (Classes, Métodos e Atributos).
- **Interface Gráfica:** Tkinter (Interface moderna e responsiva).
- **Banco de Dados:** SQLite3 (Tabelas relacionadas e comandos SQL).
- **Controle de Versão:** Git/GitHub (Histórico de commits e documentação).

## 📈 Diferenciais do Projeto
- **Cálculo de Parcelamento:** Lógica inteligente de juros (3% para planos acima de 10x).
- **Navegação Dinâmica:** Telas que se adaptam conforme o status de login do usuário.
- **Persistência:** Todos os dados são salvos localmente, permitindo que o progresso do aluno não seja perdido.
